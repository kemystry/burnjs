Class: Burn.Controller

Defined in: src/controller.coffee

Overview

Examples:

Extend Burn.Controller and register the new controller 

class CustomController extends Burn.Controller

  routes:
    'alert/:title' : 'alert'

  alert: (params) ->
    alert(params.title)

Burn.registerController(CustomController)
